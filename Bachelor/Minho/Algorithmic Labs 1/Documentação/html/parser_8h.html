void juntarStrings ( stack *  dados)

Funcao responsavel juntar Strings.

Esta funcao existe pelo simples facto que o StrTok pode cortar uma string baseada no limite = " ". Isto causaria enormes problemas pois "Ola Mundo" seria considerado 2 strings, então esta função: Sempre que encontra uma string só com " no inicio, vai percorrer a stack até encontrar uma string com " no fim Junta tudo e forma as strings pretendidas
